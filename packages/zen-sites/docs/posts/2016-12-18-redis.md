# 环境配置
1. 源码安装 redis
```shell
   # 
   wget http://download.redis.io/redis-stable.tar.gz
   tar xvzf redis-stable.tar.gz
   cd redis-stable
   make 
```
2. 安装 tcl 工具,[tcl](http://www.linuxfromscratch.org/blfs/view/cvs/general/tcl.html) 安装网页
参照连接进行安装。
3. 启动
```bash
# mac
sudo brew services start redis 
```
4. 测试
```bash
# 连接 redis
# 若正常连接说明开启成功
redis-cli
```
